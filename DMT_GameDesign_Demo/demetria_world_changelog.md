## v1.2.9 — ปุ่มแว่นขยายดูแผนที่ใหญ่ ✅ (ล่าสุด)
**เพิ่มปุ่ม 🔍 ที่มินิแมพ กดแล้วดูแผนที่เวอร์ชันใหญ่ได้**

- ปุ่มแว่นขยายอยู่มุมล่างขวาของมินิแมพ (ไม่บังตัวอักษร N/S/E/W) กดเปิด modal `#mapModal` โชว์แผนที่ขนาดใหญ่ (สูงสุด 380px ปรับลดตามขนาดจอ/มือถือ)
- แผนที่ใหญ่อัปเดตสดทุกเฟรมเหมือนมินิแมพตัวเล็ก (ตำแหน่งผู้เล่น, จุดด่านปลดล็อคแล้ว/ยังไม่ปลดล็อค, วงแหวนทองชี้ด่านเป้าหมาย) — เดินต่อได้ระหว่างเปิดแผนที่ค้างไว้ดูก็ยังเห็นตัวเองขยับ
- รีแฟกเตอร์: ดึงโค้ดวาดมินิแมพออกมาเป็น `drawMinimapTo(canvas, size)` ใช้ร่วมกันทั้งมินิแมพตัวเล็ก (132px) กับแผนที่ใหญ่ ไม่มีโค้ดวาดซ้ำ 2 ชุด
- กด ESC ปิดแผนที่ใหญ่ได้เหมือน modal อื่นๆ ในเกม

## v1.2.8 — Dev Panel: ปลดล็อคด่านได้ ✅
**เพิ่มเครื่องมือปลดล็อคด่านในแผง Dev Tools 2 แบบ**

- **⏭️ ปลดล็อคด่านถัดไป** — ปุ่มลัด ปลดล็อคด่านที่ `nextUnlockStage()` ชี้อยู่ทันที ไม่ต้องเช็คของ/เงิน/ระยะทาง เหมาะกับไล่เทสระบบตามลำดับปกติ
- **ช่องพิมพ์ชื่อด่าน** (autocomplete จาก `G.stages` ทั้งหมด) + ปุ่ม "🔓 ปลดล็อคด่านนี้" — เลือกปลดล็อคด่านไหนก็ได้ตามใจ ข้ามลำดับ chain ได้ (เผื่ออยากเทสด่านท้ายๆ โดยไม่ต้องไล่ปลดล็อคทีละด่าน)
- ทั้งสองปุ่มตั้ง `s.unlocked=true` ตรงๆ ไม่หักของ/เงิน ไม่เช็ค `nearEnough()` — เป็นเครื่องมือทดสอบล้วนๆ แยกจาก `doUnlock()` ที่ผู้เล่นใช้จริงซึ่งยังเช็คครบทุกเงื่อนไขเหมือนเดิม

## v1.2.5 — ระบุจำนวนตอนคราฟได้ ✅
**เพิ่มตัวปรับจำนวนก่อนกดคราฟในหน้าต่างร้าน NPC**

- แต่ละสูตรคราฟมีตัวปรับจำนวน (ปุ่ม − / ช่องพิมพ์ตัวเลข / ปุ่ม +) ตั้งได้ 1–99 ก่อนกดปุ่ม "คราฟ"
- วัตถุดิบที่โชว์ (`ri`) และเงื่อนไข พอ/ไม่พอ คูณตามจำนวนที่ตั้งไว้ ปุ่มคราฟจะเขียนกำกับ "×N" เมื่อตั้งมากกว่า 1
- `doCraft()` หักวัตถุดิบและเพิ่มของที่คราฟตามจำนวนในคลิกเดียว (เดิมคราฟได้ทีละ 1 เท่านั้น)
- จำนวนที่ตั้งไว้เก็บไว้ที่ `s.qty[i]` บน stage object เอง ต่อสูตรต่อร้าน อยู่จนกว่าจะรีเฟรชหน้า
- **หมายเหตุ:** ยังไม่ได้ทำใน `eco_demo.html` เพราะพี่แยมยังไม่ได้ขอ ถ้าต้องการให้ sync ด้วยบอกได้

**แยกแผงเควส (กด Q) เป็น 2 ก้อนชัดเจน เหมือนที่ทำใน `eco_demo.html` v0.7.2**

## v1.0.0 — Isometric World Release ✅
**เกมไอโซเมตริกเดินได้จริงตัวแรก ต่อยอดจากดีไซน์ ECO System 63 ด่าน**

- **World Map**: สร้างแมพ 5 โซนตามเข็มทิศ (เมืองหลวงตรงกลาง / เหนือ=หิมะ / ออก=ฟาร์ม / ใต้=ลาวา / ตก=เหมืองแร่) ด้วย value noise ให้ขอบโซนเป็นธรรมชาติไม่กลมเป๊ะ พร้อมถนนเชื่อมเมืองหลวงออกไปทั้ง 4 ทิศ
- **วางด่านอัตโนมัติ**: ดึงข้อมูล 63 stage จาก `eco_demo.html` (ITEMS + DEF + zone) มาวางบนกริดตามโซนจริง ไม่ทับกัน ทุกด่านเดินถึงได้ (verify ด้วย flood-fill)
- **การเดิน**: ตัวละครเดิน WASD/ลูกศร ชนอาคารไม่ได้ (collision), กล้องตามตัวละครแบบ smooth
- **กลางวัน-กลางคืน**: ท้องฟ้าไล่สีตามเวลา, พื้นมืดลงตอนกลางคืน, หน้าต่างอาคารที่ปลดล็อคแล้วติดไฟเหลืองพร้อมแสงกองรอบๆ, ตัวละครถือโคมส่องทางตอนกลางคืน, ปุ่มปรับความเร็วเวลา 1x/3x/8x + pause
- **Inventory**: แผงคลังของ (กด I) แสดงไอเทมที่เก็บมาได้ทั้งหมด
- **Quest**: NPC ที่ปลดล็อคแล้วสุ่มออกเควสขอไอเทม (กด Q ดู) ปุ่มเควสเรืองทองเมื่อมีเควสที่ส่งได้ ส่งเควสได้ทั้งจากแผง Quest และจากในร้าน NPC โดยตรง
- **โต้ตอบด่าน** (กด E ใกล้อาคาร หรือคลิกอาคาร): ปลูก/รอ/เก็บเกี่ยวสำหรับ passive stage, คราฟตามสูตรสำหรับ craft stage, ดูเงื่อนไขปลดล็อคพร้อม cost breakdown
- **มินิแมพ**: มุมขวาบน หมุน 45° เป็นรูปเพชรพร้อมป้ายทิศ N/S/E/W, จุดชมพู=ตัวเรา, จุดทอง=อาคารปลดล็อคแล้ว, จุดเทา=ยังล็อค
