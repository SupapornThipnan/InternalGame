## v0.7.3 — เควสรับจ้างไม่ขอของที่ร้านตัวเองคราฟ ✅ (ล่าสุด)
**กันไม่ให้ร้าน NPC ออกเควสขอของที่ตัวเองผลิตอยู่แล้ว**

- `generateQuest()` สุ่มร้อง NPC ก่อน แล้วกรอง `pool` ให้ตัด item ที่อยู่ใน `recipes` ของร้านนั้นออก (เฉพาะของที่คราฟเอง ไม่แตะวัตถุดิบที่ใช้คราฟ) ก่อนสุ่มของที่จะขอ
- ถ้าร้านไหนกรองแล้วพูลว่าง (เช่น ยังไม่มีของอื่นให้ขอเลย) จะข้ามไปลองร้านถัดไปแบบสุ่ม ถ้าไม่มีร้านไหนมีพูลพอ รอบนั้นจะไม่ออกเควส (ไม่ error ไม่ค้าง)

## v0.7.2 — เควส 2 ประเภท: เควสหลัก + เควสรับจ้าง ✅
**แยกเควสในแผงเควสเป็น 2 ก้อนชัดเจนตามที่คุยกันไว้**

- เพิ่มแผง **🎯 เควสหลัก** ด้านบนสุด — โชว์ด่านล็อคถัดไป (`G.stages.find(s=>!s.unlocked)`) พร้อมของ/เงินที่ต้องใช้ กดปลดล็อคได้ตรงนั้นเลย (เรียก `unlock()` เดิม) — ไม่ได้เงินเป็นรางวัล ปลดล็อคด่านคือรางวัล
- แผง **📜 เควสรับจ้าง** ด้านล่าง = ระบบเควสสุ่มขอของเดิมทั้งหมด (ไม่แตะโค้ดเลย) แค่เปลี่ยนหัวข้อให้ชัดว่าเป็นเควสได้เงิน
- ฟังก์ชันใหม่ `renderMainQuest()` อ่านข้อมูลสดจาก stage ปัจจุบันทุกครั้งที่ `render()` ไม่มี state แยก ไม่ชนกับระบบเควสสุ่มเดิม
- เพิ่ม CSS `.qcard.main` (ขอบสีม่วง) ให้แยกจากการ์ดเควสรับจ้าง (ขอบทอง) ด้วยสายตา

## v0.1.0 — ECO System Demo (Base)
**ระบบหลักครบ 50 ด่าน**

**ด่านทั้งหมด 50 ด่าน แบ่งเป็น 9 Wave:**
- 🌾 Farm (passive) — ปลูกและเก็บเกี่ยวไอเทม
- 🌲 Forest (passive) — รวบรวมทรัพยากรธรรมชาติ
- 🐄 Animal (passive) — เลี้ยงสัตว์ผลิตไอเทม
- 🏪 Craft NPC — คราฟไอเทมจาก recipe

**ระบบที่มี:**
- วงจรกลางวัน/กลางคืน พร้อม clock bar
- ปุ่มควบคุมความเร็ว 1x / 2x / 5x / 10x และ pause
- ปุ่มเปิด/ปิด effect แสงกลางคืน
- ระบบ inventory แสดงไอเทมที่มี
- ระบบเก็บเกี่ยว passive stage (กด 🌱 ปลูก → รอวัน → กด เก็บ)
- ระบบ craft NPC stage พร้อมเช็ค ingredient
- ระบบปลดล็อคด้วยไอเทม
- Filter tab: ทั้งหมด / ฟาร์ม / ป่า / สัตว์ / NPC / ปลดล็อคแล้ว
- Log panel บันทึกกิจกรรม
