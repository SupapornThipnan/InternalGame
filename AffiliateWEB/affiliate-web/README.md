# deetao Affiliate 📊 — คู่มือใช้งาน

เว็บจัดการข้อมูลสินค้า ITM + Dashboard สรุปยอดขาย/ค่าคอม + คิวคลิปรายเดือน + ติดตามความคืบหน้าโปรดักชั่น อ่าน-เขียนข้อมูลกับ Google Sheet "Nong F" ของพี่แยมโดยตรง ไม่มี server ไม่มีค่าใช้จ่าย

## เปิดใช้เว็บ

ดับเบิลคลิก `index.html` — เปิดในเบราว์เซอร์ได้เลย (มือถือก็ได้ ถ้าโยนไฟล์ขึ้น GitHub Pages)
ตอนนี้เว็บ **อ่านข้อมูลได้ทันที** (โหมดอ่านอย่างเดียว) — ถ้าอยากเพิ่ม/แก้ข้อมูลจากหน้าเว็บ ให้ทำขั้นตอน "เชื่อม Apps Script" ด้านล่างอีก 5 นาที

## เชื่อม Apps Script (ทำครั้งเดียว — เพื่อให้เว็บเขียนข้อมูลลง Sheet ได้)

1. เปิด [Google Sheet "Nong F"](https://docs.google.com/spreadsheets/d/1ZBFUpDBsCqTXxh6WkGwjBy1aeRBuIRQ4nc4NVEo-Xx4/edit)
2. เมนูบน: **Extensions → Apps Script** (ส่วนขยาย → Apps Script)
3. ลบโค้ดเดิมในหน้าต่างที่เปิดมา แล้ว copy โค้ดทั้งหมดจากไฟล์ `apps-script.gs` ไปวาง → กด 💾 บันทึก
4. ที่แถบด้านบน เลือกฟังก์ชัน **setup** → กด **Run** ▶️
   - Google จะขอสิทธิ์ → กด **Review permissions** → เลือกบัญชีพี่แยม
   - ถ้าเจอหน้าเตือน "Google hasn't verified this app" → กด **Advanced → Go to (unsafe)** (โค้ดของเราเอง ปลอดภัยครับ)
   - รันเสร็จ ชีต `สินค้า` และ `ยอดขาย` จะถูกสร้างในไฟล์ให้อัตโนมัติ ✅ (ชีตอื่นๆ ที่ใช้ในเว็บ เช่น 🗂️ ListITME, TODOLIST, ⚡️Progress พี่แยมสร้าง/ดูแลเองอยู่แล้ว)
5. กด **Deploy → New deployment**
   - กดไอคอน ⚙️ ข้าง "Select type" → เลือก **Web app**
   - Execute as: **Me** / Who has access: **Anyone** → กด **Deploy**
6. copy **Web app URL** (ลงท้ายด้วย `/exec`)
7. เปิดเว็บ `index.html` → แท็บ **⚙️ ตั้งค่า** → วาง URL → กด 💾 บันทึก → กด 🔌 ทดสอบการเชื่อมต่อ

เสร็จแล้ว! ทุกปุ่มที่เขียนข้อมูลกลับ Sheet จะใช้งานได้เลย

## ⚠️ ถ้าแก้โค้ด Apps Script ภายหลัง

ต้องกด **Deploy → Manage deployments → ✏️ Edit → Version: New version → Deploy**
ไม่งั้นเว็บจะยังเรียกโค้ดเวอร์ชั่นเก่าอยู่ (คนพลาดตรงนี้กันเยอะครับ) — ยกเว้นการแก้ `onEdit` (ดูหัวข้อ Sync EP อัตโนมัติ) ที่ไม่ต้อง Deploy ใหม่ แค่บันทึกโค้ดก็พอ

## แท็บต่างๆ ในเว็บ

### 📊 Dashboard
สรุปภาพรวม: การ์ดค่าคอมรวม/ยอดขายรวม/จำนวนสินค้า/EP ทั้งหมด, กราฟค่าคอมรายเดือน, สัดส่วนตามหมวดหมู่, Top 5 สินค้าทำเงิน, ตัวกรองช่วงเวลา (เดือนนี้/3 เดือน/ทั้งหมด)

### 📦 สินค้า
ตารางสินค้าทั้งหมด (ค้นหา/กรอง/sort ได้ทุกคอลัมน์) + ปุ่มเพิ่ม/แก้ไขสินค้า + ปุ่ม 🆕 เพิ่มสินค้าใหม่ (ครบวงจร — เขียนสินค้า+⚡️Progress+🗂️ ListITME+คำอธิบาย/ShopeeLink พร้อมกัน) ส่งเข้า Google Sheet ผ่าน Apps Script
**หมายเหตุ: มีปุ่มลบ 🗑️ เฉพาะตารางสินค้าเท่านั้น** (มี popup ถามยืนยันก่อนเสมอ ลบแค่ชีต `สินค้า` ไม่กระทบชีตอื่น) ใช้สำหรับล้างแถวที่กดซ้ำ/ผิดพลาด — ปกติอยากเลิกขายสินค้าให้เปลี่ยนสถานะเป็น "เลิกขาย" แทนการลบ

### 🔍 ค้นหา
ค้นหาสินค้าด้วยรหัส ITM_XXX / ชื่อ / คำในคำอธิบาย ผลลัพธ์แต่ละรายการโชว์:
- แท็ก Progress สี (Scripts/Sounds/Footage/Draft/Final) — คลิกแล้วเด้งไปหน้า ✍️ Progress พร้อมค้นหาไอเทมนั้นให้เลย
- คำอธิบายพร้อมปุ่มก๊อป, ลิงก์ Shopee ทุกอันพร้อมปุ่มก๊อปแยกแต่ละอัน

### 📅 คิวคลิป
หน้าหลักที่พี่แยมใช้ทำงานประจำวัน — ดึงข้อมูลจากชีต `TODOLIST-<เดือน>` เดือนปัจจุบัน (เลือกชื่อชีตได้ในแท็บตั้งค่า) มาแสดงเป็นตาราง เรียงตามวันที่:

- **สถิติด้านบน**: จำนวนคลิปที่ลง TikTok แล้วเดือนนี้ / เป้าหมายคลิปต่อเดือน (ตั้งเองได้) / วันที่เหลือในเดือน / ค่าเฉลี่ยคลิปที่ต้องลงต่อวันให้ถึงเป้า
- **คอลัมน์ Scripts / Sounds / Footage / Final**: checkbox ติ๊กได้อิสระ ไม่ผูกกับอะไรทั้งนั้น — กดซ่อน/แสดงกลุ่มนี้ได้ทั้งตาราง (จำสถานะไว้ข้ามการเปิดเว็บใหม่)
- **คอลัมน์ Backup**: checkbox แยกต่างหาก ผูกกับคอลัมน์ "UpSpace" ในชีต (สำรองไฟล์ขึ้นคลาวด์แล้วหรือยัง) ไม่เกี่ยวกับการลงคลิปเลย
- **คอลัมน์ TikTok**: กด "✅ ลง TikTok แล้ว" → ต้องใส่ลิงก์คลิป → ระบบติ๊ก Scripts/Sounds/Footage/Final/List ให้ครบอัตโนมัติ + sync EP เข้า 🗂️ ListITME และ ⚡️Progress (Final) ให้เอง กดที่ลิงก์ที่ขึ้นมาแล้วเพื่อแก้ลิงก์หรือกด "ยังไม่ได้ลงคลิป" (ลด EP ลง 1 คืนสถานะทุกอย่างกลับ) ได้
- **คอลัมน์ Shopee**: กด "✅ ลง Shopee แล้ว" → ใส่ลิงก์ หรือกด "ร่าง" (บันทึกว่ากำลังทำอยู่ ยังไม่ติ๊กจริง) หรือกด "ไม่มีลิงก์" (สำหรับไอเทมที่ไม่ได้ขายบน Shopee เลย — เอา checkbox ออกจาก Sheet ไปเลย กดซ้ำเพื่อ "ยกเลิกไม่มีลิงก์" คืน checkbox ได้) — ไม่ผูกกับ EP หรือ Scripts/Sounds/Footage/Final ใดๆ
- **ปุ่ม "🛍️ คิวลง SHOPEE VDO"**: หาไอเทมตัวถัดไปที่ต้องลง Shopee ให้ (ข้ามตัวที่ลงแล้ว/ไม่มีลิงก์ นับตัวที่ "ร่าง" ไว้ด้วย) เด้งป็อปอัพบอกชื่อ+EP
- **คลิกที่ชื่อไอเทม (ITM_xxx + ชื่อ)**: ก๊อปคำอธิบายสินค้า + เปิดป็อปอัพข้อมูลเต็ม (เหมือนการ์ดในหน้า 🔍 ค้นหา — แท็ก Progress, คำอธิบาย, ลิงก์ Shopee ทุกอัน) โดยไม่ต้องออกจากหน้าคิวคลิป คลิกที่แท็ก Progress ในป็อปอัพนี้เพื่อไปหน้า ✍️ Progress พร้อมค้นหาไอเทมนั้นได้เลยเหมือนหน้าค้นหา ในป็อปอัพยังมีปุ่ม **"#SHOPEE"** สำหรับก๊อปข้อความแฮชแท็กมาตรฐานสำหรับแปะโพสต์ Shopee ด้วย (ตั้งข้อความเองได้ในแท็บตั้งค่า)

### ✍️ Progress
กรอกความคืบหน้าโปรดักชั่นเข้าชีต `⚡️Progress` เอง (Scripts/Sounds/Footage/Draft เป็น dropdown เลือกเลข EP, Final แก้จากหน้านี้ไม่ได้ — sync มาจากการกดลง TikTok ในคิวคลิปเท่านั้น) แก้กี่ช่องก็ได้ กดปุ่ม 💾 บันทึกทีเดียวจบ

### ⚙️ ตั้งค่า
- เชื่อมต่อ Apps Script Web App URL
- เลือกชื่อชีต TODOLIST ของเดือนปัจจุบัน (ให้แท็บคิวคลิปดึงมาแสดง)
- ตั้งข้อความแฮชแท็กสำหรับปุ่ม "#SHOPEE" (อยู่ในป็อปอัพข้อมูลไอเทม แท็บคิวคลิป)

## 🔄 Sync อัตโนมัติต่างๆ

- **TODOLIST → ListITME**: พอพี่แยมติ๊กช่อง **List** ในชีต TODOLIST (หรือกดลง TikTok ในคิวคลิปซึ่งติ๊กให้อัตโนมัติ) ระบบจะอัปเดตค่า EP ปัจจุบันใน `🗂️ ListITME` ให้ตรงกับ EP ที่ใช้ในคลิปนั้นทันที เป็น `onEdit` trigger — **ไม่ต้อง Deploy ใหม่** แค่บันทึกโค้ดใน Apps Script ก็ทำงานได้เลย (คนละกลไกกับ doGet/doPost)
- **ลง TikTok → ⚡️Progress (Final)**: กดลง TikTok แล้ว จะดัน EP เข้าคอลัมน์ Final ของ ⚡️Progress ให้อัตโนมัติด้วย (เป็นค่าเดียวที่ sync อัตโนมัติจากคิวคลิป — Scripts/Sounds/Footage/Draft พี่แยมกรอกเองผ่านหน้า ✍️ Progress)
- **Progress (Footage/Draft) → ListITME**: แก้ Footage หรือ Draft ในหน้า ✍️ Progress จะ sync ค่าเดียวกันเข้า `🗂️ ListITME` คอลัมน์ Footage/Draft ให้ด้วย

## โครงสร้างข้อมูลใน Sheet "Nong F"

| ชีต | ใช้ทำอะไร | ดูแลโดย |
|-----|-----------|---------|
| `สินค้า` | รายการสินค้า SKU/ราคา/ค่าคอม/สถานะ | เว็บ (เพิ่ม/แก้ได้) |
| `ยอดขาย` | บันทึกยอดขาย/ค่าคอมต่อวัน | เว็บ (เพิ่มได้) |
| `🗂️ ListITME` | EP ปัจจุบันของแต่ละไอเทม แบ่งตาม Tier (A/B/B2/Bต/C/Cต) | พี่แยมแก้เอง + เว็บ sync EP ให้ |
| `📝 TODOLIST-<เดือน>` | คิวคลิปที่ต้องถ่าย/ลงในเดือนนั้น (ชื่อเปลี่ยนทุกเดือน) | พี่แยมวางแผนเอง + เว็บติ๊ก/บันทึกลิงก์ให้ |
| `⚡️Progress` | EP ล่าสุดของแต่ละ stage การผลิต (Scripts/Sounds/Footage/Draft/Final) ต่อไอเทม | หน้า ✍️ Progress |
| `📝#คำอธิบาย` | ข้อความคำอธิบาย/แฮชแท็กของแต่ละไอเทม | พี่แยมกรอกเอง |
| `🍊🔗ShopeeLink` | ลิงก์ Shopee + หมวดหมู่ของแต่ละไอเทม | พี่แยมกรอกเอง |

ถ้า Sheet จริงของพี่แยมมีคอลัมน์ไม่ตรงกับที่โค้ดคาดไว้ **ให้ยึดตาม Sheet จริงเสมอ** — ดึงข้อมูลแถวแรก (header) มาดูก่อน แล้วแก้โค้ดตามคอลัมน์ที่มีอยู่จริง (คอลัมน์ TODOLIST หลายอันอ้างอิงด้วยตำแหน่ง ไม่ใช่ชื่อ header — ดู `apps-script.gs` บนสุดของไฟล์)

หมายเหตุ: เว็บตั้งใจไม่มีปุ่มลบข้อมูลไว้แต่แรก (กันพลาด) — ยกเว้นปุ่ม 🗑️ ในตารางสินค้าเท่านั้น (มี popup ยืนยันก่อนเสมอ) ส่วนที่เหลือ ทุกอย่างที่ "ยกเลิก" ได้ (เช่น ยกเลิกลง TikTok/Shopee, ยกเลิกไม่มีลิงก์) จะคืนค่าเดิม ไม่ใช่ลบแถว
