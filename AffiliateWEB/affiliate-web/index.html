    ⚙️ เชื่อมต่อ Apps Script (เพื่อเพิ่ม/แก้ข้อมูลจากเว็บ)

    1. เปิด Google Sheet ของพี่แยม → Extensions → Apps Script
    2. วางโค้ดจากไฟล์ apps-script.gs → รันฟังก์ชัน setup หนึ่งครั้ง (สร้างชีตให้อัตโนมัติ)
    3. Deploy → New deployment → Web app → Execute as Me / Access Anyone
    4. คัดลอก Web app URL มาวางด้านล่าง (ดูละเอียดใน README.md)

    📅 คิวคลิป (ชีต TODOLIST เดือนนี้)

    พิมพ์หรือเลือกชื่อชีต TODOLIST ของเดือนปัจจุบัน เพื่อให้แท็บ "📅 คิวคลิป" ดึงรายการมาแสดง — กด "โหลดรายชื่อชีต" เพื่อดึงชื่อชีตทั้งหมดมาช่วยเลือก (ต้องเชื่อม Apps Script ด้านบนก่อน)

    🏷️ ข้อความแฮชแท็ก Shopee (ปุ่ม "#SHOPEE" ในป็อปอัพข้อมูลไอเทม แท็บคิวคลิป)

    ข้อความที่จะถูกคัดลอกเวลากดปุ่ม "#SHOPEE" ในป็อปอัพที่เปิดจากไอคอน 📋 ข้างชื่อไอเทม แท็บ 📅 คิวคลิป
