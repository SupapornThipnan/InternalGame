## v0.6.1 — Triple Crafted Unlock Costs ✅ (ล่าสุด)
**เพิ่มของคราฟในเงื่อนไขปลดล็อคอีก 2 ชิ้นต่อด่าน (Wave 2–9)**

ทุกด่านตั้งแต่ Wave 2 เป็นต้นไปมีของคราฟใน `unlockCost` เพิ่มขึ้นจาก 1 ชิ้น (v0.6.0) เป็น **3 ชิ้น** โดยเลือกจากไอเทมคราฟที่ได้จาก NPC stage ก่อนหน้าในลำดับ array:

| Wave | ตัวอย่างของคราฟที่ใช้ปลดล็อค (3 ชิ้น) |
|------|-----------------------------------------|
| W2 | ขนมปัง + แยมเบอร์รี่ + สลัด |
| W3 | ซุปเห็ด + ของเล่นไม้ + เฟอร์นิเจอร์ |
| W4 | กาแฟ / ชา / เค้ก + อุปกรณ์ฟาร์ม / เครื่องมือ |
| W5 | สมูทตี้ + มัฟฟิน/ทาร์ต/น้ำมะนาว/น้ำส้ม/ไอศครีม |
| W6 | ราเมน / พิซซ่า / ไอศครีม + ชีส / ขนมปังกระเทียม / ไข่ต้มยาง |
| W7 | ยาบำรุง + แหวน/ต่างหู/ผ้าห่ม/สร้อย/กระเป๋า |
| W8 | เทียนหอม + หนังสือ/รูปปั้น/รูปวาด/สบู่/แผนที่ |
| W9 | เครื่องสาย + โน้ตเพลง / กระถาง + ยาสัตว์ |

## v0.6.0 — Crafted Unlock Costs ✅
**ปรับเงื่อนไขปลดล็อคให้ต้องใช้ของคราฟจาก NPC ด่านก่อนหน้า**

เพิ่มไอเทมคราฟเข้า `unlockCost` ของทุกด่าน Wave 2–9 ให้ต้องนำของที่คราฟจาก NPC stage ก่อนหน้ามาใช้ปลดล็อคด้วย:

| Wave | NPC ที่ผลิต | ตัวอย่างของที่ต้องใช้ปลดล็อค |
|------|-------------|-------------------------------|
| W2 | ร้านขนมปัง | ขนมปัง, แยมเบอร์รี่, สลัด |
| W3 | ร้านซุป, โรงงานไม้ | ซุปเห็ด, เฟอร์นิเจอร์, ของเล่นไม้ |
| W4 | ร้านกาแฟ, โรงตีเหล็ก | กาแฟ, เค้ก, อุปกรณ์ฟาร์ม, เครื่องมือ |
| W5 | ร้านของหวาน, ร้านน้ำผลไม้ | ไอศครีม, สมูทตี้, มัฟฟิน, น้ำมะนาว |
| W6 | ร้านราเมน, ร้านพิซซ่า | ราเมน, พิซซ่า, ไอศครีม |
| W7 | ร้านสมุนไพร, ร้านเครื่องประดับ, ร้านตัดเย็บ | ยาบำรุง, แหวน, สร้อย, เสื้อผ้า, กระเป๋า |
| W8 | โรงทำเทียน, ร้านศิลปะ, ร้านหนังสือ | สบู่, รูปปั้น, รูปวาด, หนังสือ, แผนที่ |
| W9 | คลินิกสัตว์, โรงดนตรี | ยาสัตว์, โน้ตเพลง |

## v0.5.2 — Quest Pool Safety ✅
**ดักบัค: เควสขอของที่ยังทำไม่ได้**

- Quest pool กรองเฉพาะของคราฟที่ ingredient ครบจาก stage ที่ปลดล็อคแล้วเท่านั้น
- ใช้ while-loop handle chain crafting (เช่น ซุปกระดูก → ราเมน)
- สตูว์ / ต้มยำ จะไม่ขึ้นเควสจนกว่าจะปลดล็อคโรงกระต่าย (เนื้อกระต่าย)

## v0.5.0 — Quest Pool Enhancement
**เพิ่มของคราฟเข้า Quest Pool**

- เควสสามารถขอของคราฟ (จาก NPC stage) ได้นอกจากวัตถุดิบธรรมดา

- ของคราฟให้ reward ต่อชิ้นสูงกว่า: **15–25 💰** vs **8–15 💰**

## v0.4.0 — Stage Unlock Money Cost
**เพิ่มค่าใช้จ่ายเงินในการปลดล็อคด่าน**

- ปลดล็อคด่านต้องใช้ทั้ง **ไอเทม + เงิน** (ยกเว้น 4 ด่านแรก ฟรี)
- ค่าเงินปลดล็อค scale ตาม wave:

  | Wave | ราคา |
  |------|------|
  | 1–4 ด่านแรก | ฟรี |
  | Wave 2 | 💰 150 |
  | Wave 3 | 💰 300 |
  | Wave 4 | 💰 600 |
  | Wave 5 | 💰 1,000 |
  | Wave 6 | 💰 1,500 |
  | Wave 7 | 💰 2,500 |
  | Wave 8 | 💰 4,000 |
  | Wave 9 | 💰 6,000 |

- การ์ดด่านแสดง chip เงินที่ต้องใช้ (🔴 ไม่พอ / 🟢 พอ)
- ปุ่มแจ้งแยก "🔒 เงินไม่พอ" กับ "🔒 ของยังไม่พอ"

## v0.1.0 — ECO System Demo (Base)
**ระบบหลักครบ 50 ด่าน**

**ด่านทั้งหมด 50 ด่าน แบ่งเป็น 9 Wave:**
- 🌾 Farm (passive) — ปลูกและเก็บเกี่ยวไอเทม
- 🌲 Forest (passive) — รวบรวมทรัพยากรธรรมชาติ
- 🐄 Animal (passive) — เลี้ยงสัตว์ผลิตไอเทม
- 🏪 Craft NPC — คราฟไอเทมจาก recipe

**ระบบที่มี:**
- วงจรกลางวัน/กลางคืน พร้อม clock bar
- ปุ่มควบคุมความเร็ว 1x / 2x / 5x / 10x และ pause
- ปุ่มเปิด/ปิด effect แสงกลางคืน
- ระบบ inventory แสดงไอเทมที่มี
- ระบบเก็บเกี่ยว passive stage (กด 🌱 ปลูก → รอวัน → กด เก็บ)
- ระบบ craft NPC stage พร้อมเช็ค ingredient
- ระบบปลดล็อคด้วยไอเทม
- Filter tab: ทั้งหมด / ฟาร์ม / ป่า / สัตว์ / NPC / ปลดล็อคแล้ว
- Log panel บันทึกกิจกรรม
